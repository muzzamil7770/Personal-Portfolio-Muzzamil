## 🔄 GitHub Actions & CI/CD

This repository uses GitHub Actions for automated deployment to Netlify.

### Workflow Features:
- 🚀 **Automatic Deployment** - Deploys to Netlify on every push to `main` branch
- ✅ **Build Validation** - Validates HTML/CSS before deployment
- 🔔 **Status Notifications** - Notifies on deployment success/failure
- 🏷️ **Preview URLs** - Generates unique preview links for each deployment

### Workflow Triggers:
- `push` to `main` branch
- `pull_request` to `main` branch
- Manual trigger via GitHub Actions UI

### Environment Variables Required:
| Variable | Description | Where to Set |
|----------|-------------|--------------|
| `NETLIFY_AUTH_TOKEN` | Netlify API token | GitHub Secrets |
| `NETLIFY_SITE_ID` | Netlify site ID | GitHub Secrets |

### How to Set Up:
1. Go to your Netlify dashboard
2. Navigate to **User Settings > Applications > Personal access tokens**
3. Create a new token and copy it
4. Go to GitHub repo > **Settings > Secrets and variables > Actions**
5. Add `NETLIFY_AUTH_TOKEN` and `NETLIFY_SITE_ID` as repository secrets
